AU Material Storage

This application reads live account data from the Guild Wars 2 API. It requires an account API key with appropriate permissions.

Nothing is uploaded. Nothing is stored remotely. If you uninstall this app, your locally stored key is deleted alongside it.

Do not share your API key with anyone else. This key is unique to your account. If in the wrong hands, it could be used maliciously. UwU.
